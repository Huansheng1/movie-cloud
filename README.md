# 云开发打造 豆瓣电影

## 如何使用？

1. 克隆到本地：`git clone git@github.com:Huansheng1/movie-cloud.git`
2. 下载安装 微信开发者工具
3. 导入项目
4. 开发云服务
5. 鼠标在项目目录右键：在终端打开
6. 命令行输入：`npm install`
7. 云服务目录邮件上传至服务器
8. 点击工具栏 - 编译
